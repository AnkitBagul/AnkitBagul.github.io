Ankit Bagul's Portfolio Website
Overview
This is the repository for my personal portfolio website, showcasing my resume, skills, projects, blogs, social media links, LinkedIn profile, research papers, and more.
Features
* Resume: View and download my resume.
* Skills: Browse through my skills and expertise.
* Projects: Explore my projects with descriptions and links.
* Blogs: Read my blog posts on various topics.
* Social Media: Follow me on social media platforms.
* LinkedIn: Connect with me on LinkedIn.
* Research Papers: Access my research papers and publications.
Technologies Used
* HTML
* CSS
* JavaScript
* Bootstrap (optional)
* React (optional)
* Node.js (optional)
Getting Started
To run this project locally, follow these steps:
1. Clone the repository: git clone https://github.com/your-username/your-repo.git
2. Navigate to the project directory: cd your-repo
3. Open the index.html file in your web browser.
Deployment
This website is deployed using Vercel at www.ankitbagul.in.
